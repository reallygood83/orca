# Orca 조율 스튜디오 — 사용 방법 (배포용)

**공개 사이트:** https://orca-lime.vercel.app  
**소스:** https://github.com/reallygood83/orca  

이 도구는 **서버에 설정을 저장하지 않습니다.**  
각 사용자가 사이트에서 자기 모드를 만들고, **자기 PC(Orca)** 에 설치합니다.

---

## 당신(운영자)이 할 일 — 배포

| 단계 | 내용 |
|------|------|
| 1 | 이미 Vercel이 `main`에 연결돼 있으면 **푸시만 하면 자동 배포** |
| 2 | 사람들에게 **사이트 링크만 공유** |
| 3 | (선택) 유튜브/오픈채팅에 아래 **복붙 안내문** 사용 |

### 공유 링크 (이 한 줄이면 됨)

```text
Orca 조율 모드 만들기 (무료·설치 가이드 포함)
https://orca-lime.vercel.app
```

```text
여러 AI를 팀장–실무로 나눠 쓰는 「조율 모드」를 직접 만들 수 있는 사이트입니다.

1) 접속: https://orca-lime.vercel.app
2) 탭 1–2로 개념 확인
3) 탭 3에서 팀장·실무 모델 고르기

※ 각자 자기 컴퓨터에 설치합니다. 제 PC 설정이 필요 없습니다.

```

### Vercel 확인

1. [Vercel 대시보드](https://vercel.com) → 프로젝트 `orca` (또는 해당 이름)  
2. Production URL = `https://orca-lime.vercel.app`

새 프로젝트라면:

1. Vercel → Import `reallygood83/orca`

## 일반 사용자(구독자)가 할 일 — 5분 흐름

### 준비물

- [Orca](https://orca.com) (또는 사용 중인 Orca 앱)  
- 실무로 쓸 CLI 중 하나 이상: Codex / Claude Code / Grok CLI 등  
- (권장) 엔진 스킬 한 번 설치:

```bash
npx skills add https://github.com/stablyai/orca --skill orchestration
npx skills add https://github.com/stablyai/orca --skill orca-cli
```

### 따라 하기

1. **https://orca-lime.vercel.app** 접속  
2. **1. 이게 뭐야?** · **2. 어떻게 돌아가?** 읽기  
3. **3. 내 모드 만들기**  
   - 모드 이름 (영문 폴더명)  
   - 팀장 AI (예: Grok)  
   - 실무 모델 (예: Codex gpt-5.6 xhigh)  
   - (선택) 리뷰 워커 명령  
4. **4. 설치 방법**  
   - 「.md 다운로드」 또는 미리보기 복사  
   - 안내대로 `$HOME/.orca/<모드이름>/` 에 저장  
5. Orca → **Settings → Quick Commands**  
   - Label = 화면에 보일 이름  
   - Scope = **Global**  
   - Text = `quick-command` 문장 전체  
6. Orca에서 **Experimental → Orchestration ON**  
7. 팀장 탭에서 Quick Command 선택 → **Goal:** 뒤에 할 일 적고 실행  

### 자주 하는 실수

| 실수 | 해결 |
|------|------|
| 사이트만 보고 끝냄 | **PC에 파일 저장 + Quick Command 등록**까지 해야 함 |
| dispatch에 모델을 넣으려 함 | 모델은 **실무 실행 명령**에 이미 들어 있음 |
| 다른 사람 모드 파일을 찾음 | **불필요** — 사이트에서 각자 생성 |
| Orchestration 실험 기능 OFF | Orca 설정에서 ON |

## 고급: CLI로 팩 만들기 (개발자)

```bash
git clone https://github.com/reallygood83/orca.git
cd orca
bash generate-pack.sh \
  --name my-orch \
  --display "MyOrch" \
  --coord grok \
  --worker codex \
  --worker-cmd 'codex -m gpt-5.6 -c model_reasoning_effort="xhigh"' \
  --max 3
```

→ `$HOME/.orca/my-orch/` 생성  

에이전트 스킬 템플릿: 저장소 `skills/orca-mode-pack/`
